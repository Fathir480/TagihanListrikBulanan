# TagihanListrikBulanan

Projek ini adalah sebuah studycase yang saya selesaikan sebagai latihan saya. Latihan ini dirancang oleh Ai sebagai berikut:

### Skenario dan Latar Belakang
Kamu baru saja direkrut sebagai developer junior di sebuah koperasi perumahan.
Bendahara koperasi masih menghitung tagihan listrik warga secara manual setiap bulannya — repot dan rawan salah.

Tugasmu: buat program CLI sederhana yang bisa menerima input pemakaian listrik (kWh) dari beberapa warga, menentukan golongan tarif secara otomatis, lalu mencetak rincian tagihan dan rekap akhir bulan.
